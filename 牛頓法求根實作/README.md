# 程式設計：第八題 - 牛頓法求根實作

實作了數值分析中的 **牛頓法 (Newton's Method)**，用於求解特定二次方程式的實根。

## 題目說明
撰寫一程式求解方程式 $f(x) = x^2 - 2x - 5$ 的根。

### 核心數學公式
* **牛頓迭代式**：
    $$x_{n} = x_{n-1} - \frac{f(x_{n-1})}{f'(x_{n-1})}$$
* **誤差判定條件**：
    $$|f(x_n) - f(x_{n-1})| \leq eps$$

### 測試參數
| 參數 | 數值 |
| :--- | :--- |
| 容許誤差 (eps) | 0.001 |
| 最大迭代次數 (Max_itera) | 100 |
| 初始點 (a) | 5 |
| 初始點 (b) | -3 |

# 程式碼逐行詳解：牛頓法求根

這份程式碼將邏輯完整平鋪，非常適合初學者理解。以下是針對每一部分的詳細拆解：

---

### 1️ 引入庫與標準空間
* `#include <iostream>`：基礎輸出入庫，讓我們能使用 `cout` 將結果印在螢幕上。
* `#include <cmath>`：數學工具庫，提供 `abs()` 函數來處理誤差的**絕對值**。
* `using namespace std;`：標準命名空間，讓我們簡寫程式碼（不必每次都寫 `std::`）。

### 2️ 主程式與參數設定
* `double eps = 0.001;`：**誤差門檻**。這是我們對精確度的要求。
* `int max_itera = 100;`：**安全煞車**。避免因為無法收斂而導致程式無限死循環。
* `double x_curr, x_next;`：分別存儲「目前的猜測值」與「計算後的下一個值」。
* `int iter;`：**計數器**，紀錄我們到底嘗試了幾次才找到答案。

---

### 3️ 核心邏輯 (以 $x_0 = 5$ 為例)
* **設定起點**：`x_curr = 5;` 告訴程式從哪裡開始找根。
* **啟動迴圈**：`for (iter = 1; iter <= max_itera; iter++)` 最多嘗試 100 次。
    * **計算新位置**：
        `x_next = x_curr - (x_curr * x_curr - 2 * x_curr - 5) / (2 * x_curr - 2);`
        這就是公式：$$x_{new} = x - \frac{f(x)}{f'(x)}$$
        > 括號內對應題目給的 $x^2 - 2x - 5$ 以及微分後的 $2x - 2$。
    * **計算函數值**：利用 `f_next` 與 `f_curr` 算出新舊兩點的 y 軸位置。
    * **判斷終止 (Break)**：
        `if (abs(f_next - f_curr) <= eps)`：如果兩次結果的差距小於 0.001，就代表答案夠準了。
    * **更新數值**：
        `x_curr = x_next;`：如果還不夠準，就把新的點當作下一輪的起點。

---

### 4️ 輸出結果
* `cout << "當 x0 = 5 時：" << endl;`：印出清楚的標示。
* `cout << "(a) root: " << x_curr << " (b) 迭代次數: " << iter << endl;`：印出最終答案。

---

### 給學生的觀念小叮嚀
> **1. 為什麼要用 `abs()`？**
> 誤差（兩數相減）可能是負值。如果不取絕對值，負數永遠會小於 `0.001`，程式會在第一步就誤判「已達標」而錯誤停止。
>
> **2. 為什麼要執行 `x_curr = x_next`？**
> 這就像是**接力賽**，必須要把這一棒算出來的最佳結果交給下一棒，當作下一次逼近的基礎，否則程式永遠只會在原點踏步。
