# 程式設計：第八題 - 牛頓法求根實作

實作了數值分析中的 **牛頓法 (Newton's Method)**，用於求解特定二次方程式的實根。

## 題目說明
撰寫一程式求解方程式 $f(x) = x^2 - 2x - 5$ 的根。

### 核心數學公式
* **牛頓迭代式**：
    $$x_{n} = x_{n-1} - \frac{f(x_{n-1})}{f'(x_{n-1})}$$
* **誤差判定條件**：
    $$|f(x_n) - f(x_{n-1})| \leq eps$$

### 測試參數
| 參數 | 數值 |
| :--- | :--- |
| 容許誤差 (eps) | 0.001 |
| 最大迭代次數 (Max_itera) | 100 |
| 初始點 (a) | 5 |
| 初始點 (b) | -3 |
